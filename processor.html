Processor

What is it and what does it do?

The processor or CPU (Central Processing Unit) for short is the heart of the computer that processes data and is responsible for all the calculations in the PC. A processor usally has multiple cores which each can handle their own task. The processor can have a graphical unit built into it too. This is not visible to the user though, but it allows the user to have no graphics card, using the so-called onboard graphics.

Where is it?

The processor sits most commonly in the middle of the motherboard. It is usually not visible, as the cooling is installed right on top of it. The cooling can be removed to access the processor. The image on the right shows the position of the processor, when the cooling has been removed.
